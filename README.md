# YouTube Analytics Dashboard 📊

This project is an interactive **YouTube Analytics Dashboard** built using **Streamlit** and the **YouTube Data API**.  
It allows users to explore channel performance and video engagement through interactive charts and visualizations.

## Features
- Search YouTube channels by **channel name**
- Display **channel statistics** (subscribers, views, total videos)
- Identify **top-performing videos**
- Analyze **likes and engagement trends**
- View **recent video thumbnails and titles**
- Interactive charts and analytics using **Plotly**

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- YouTube Data API v3

## How to Run the Project

1. Clone the repository
2. Install required libraries
3. Add your **YouTube API key** in the code
4. Run the Dashboard (streamlit run app.py)
5. The dashboard will open in your browser at: https://localhost:8501


## Project Goal
The goal of this project is to demonstrate how **real-time API data can be transformed into meaningful visual insights using Python and Streamlit**.
